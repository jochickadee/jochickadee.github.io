Week One

Command Line, Version Control and Git

November, 14th 2015

I made it through my first week of Dev Bootcamp! Huzzah! This week we learned how to use the terminal, what version control is and why it's important and how to use the Git version control system.

Version control is a way of tracking all your edits and file for a project in a way that you can: 1. Go back in time and fix errors or make changes 2. Have multiple people working on the same project without creating a mess of different files and versions of your project.
